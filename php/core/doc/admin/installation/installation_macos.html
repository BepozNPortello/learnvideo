Mac OS X¶

This relies on MAMP, which provides the required environment.

  1. Install MAMP and run it.
  2. Go to ‘Preferences‘, open the ‘Apache’ tab and set ‘Document Root’ to /Users/<YOUR USER NAME>/Sites, so your Sites directory will be used as Apache root.
  3. Download ownCloud
  4. Extract it
unzip owncloud-latest.zip -d ~/Sites/
  1. Now you can set it up by going to http://localhost:8888/owncloud
